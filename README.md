## ABPawClinic

AB Paw Klinik Hewan adalah aplikasi desktop berbasis Java dan MySQL yang digunakan untuk mengelola operasional klinik hewan secara terintegrasi. Aplikasi ini memiliki tiga jenis pengguna: Admin, Dokter, dan Pemilik Hewan.

Pemilik Hewan dapat melakukan registrasi akun beserta data hewan peliharaannya. Setelah login, pemilik dapat memilih dokter berdasarkan spesialisasi atau harga, melakukan konsultasi online. Pemilik juga bisa booking antrean offline, membeli obat (dengan/tanpa resep). Dokter memiliki fitur chat realtime dengan pasien, mengupdate status pemesanan (menunggu, diproses, selesai, batal), mengisi biaya jasa untuk pemeriksaan offline, serta membuat resep obat yang otomatis masuk ke pembelian obat pemilik. Admin bertugas mengelola data dokter dan pemilik, mengelola kupon diskon, serta melihat berbagai laporan.

Aplikasi ini memiliki fitur keamanan  yang mencakup data admin, dokter, pemilik, hewan, pemesanan online & offline, pembelian obat, resep, kupon, chat, alergi, dan detail resep.
